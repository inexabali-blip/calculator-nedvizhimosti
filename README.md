# Калькулятор доходности недвижимости

Веб-приложение, помогающее инвестору оценить доходность объекта недвижимости (вилла, апартаменты, мини-отель) с учётом аренды, операционных расходов, налогов и кредитного финансирования.

## Возможности

- ввод исходных параметров по объекту, аренде, расходам, налогам и финансированию;
- автоматический пересчёт ключевых метрик: валовая выручка, NOI, платежи по кредиту, денежный поток;
- показатели эффективности: cash-on-cash, cap rate, срок окупаемости и точка безубыточности;
- пресеты значений и кнопка сброса к значениям по умолчанию.

## Архитектура проекта

```
├─ public/                 # Статические файлы (HTML, favicon)
├─ src/
│  ├─ calculations/        # Чистые функции расчётов (без привязки к UI)
│  ├─ components/          # React-компоненты интерфейса
│  ├─ hooks/               # Хуки для работы с состоянием формы и расчётами
│  ├─ styles/              # Глобальные стили приложения
│  ├─ types/               # Общие типы входных/выходных данных
│  ├─ utils/               # Утилиты форматирования
│  └─ main.tsx             # Точка входа приложения
├─ tests/                  # Юнит-тесты для функций расчёта (Vitest)
├─ vite.config.ts          # Конфигурация Vite/Vitest
└─ tsconfig*.json          # Настройки TypeScript
```

Основная бизнес-логика изолирована в `src/calculations/rentalCalculator.ts`, что упрощает тестирование и повторное использование формул без UI.

## Требования

- Node.js 18+
- npm 9+

## Установка и запуск

```bash
npm install          # Установка зависимостей
npm run dev          # Запуск dev-сервера (http://localhost:5173)
```

### Проверка и сборка

```bash
npm run test         # Запуск юнит-тестов (Vitest)
npm run build        # Сборка production-бандла
npm run preview      # Предпросмотр собранного приложения
```

## Тесты

В каталоге `tests/` лежат сценарии `rentalCalculator.test.ts`, покрывающие базовые кейсы для помесячной и посуточной аренды. Тесты запускаются командой `npm run test`.
